INA260

	В директории /sys/class/hwmon/ должно быть 3 симлинка с именами hwmon0...hwmon2.
Вероятнее всего, "наши" девайсы это hwmon1 и hwmon2 (чтобы убедиться наверняка,
нужно распарсить пути, на которые указывают симлинки, наши девайсы должны содержать
подстроки "i2c-0/0-0040" и "i2c-0/0-0041" соответственно).

	Каждый из мониторов предоставляет 3 псевдофайла, через которые можно получить
текущие значения напряжения, тока и мощности:
 - current_monitor - текущее значение тока, мА
 - voltage_monitor - текущее значение напряжения, мВ
 - power_monitor - текущее значение потребляемой мощности, мВт

	По умолчанию, длительность одного измерения тока и напряжение - 1.1 мс, количество
измерений - 128 (при запросе выводится усредненное значение параметра). Для большенства
случаев данные параметры будут оптимальным вариантом.

	В случае необходимости изменить какой-либо из выше описанных параметров (время
измерения тока/напряжения или количество измерений), драйвер предоставляет следующие
псевдофайлы:

 - averaging_mode - количество измерений (усреднений)
На вход принимает значения от 0 до 7:
attr = 0, number of averages = 1
attr = 1, number of averages = 4
attr = 2, number of averages = 16
attr = 3, number of averages = 64
attr = 4, number of averages = 128
attr = 5, number of averages = 256
attr = 6, number of averages = 512
attr = 7, number of averages = 1024

 - voltage_conv_time - время измерения напряжения (1-го измерения)
 - current_conv_time - время измерения тока (1-го измерения)
На вход принимают значения от 0 до 7:
attr = 0, conversion time = 140 µs
attr = 1, conversion time = 204 µs
attr = 2, conversion time = 332 µs
attr = 3, conversion time = 588 µs
attr = 4, conversion time = 1.1 ms
attr = 5, conversion time = 2.116 ms
attr = 6, conversion time = 4.156 ms
attr = 7, conversion time = 8.244 ms
