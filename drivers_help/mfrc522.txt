MFRC522

	В директории /dev находится псевдофайл mfrc522_dev. Данный файл может
быть открыт в режиме чтения и записи. Методу read и write должен быть передан
буфер размером не менее 16 байт. Операция чтения блокирующая, метод вернет
результат только в случае удачного чтения метки. В случае прерывания чтения
будет возвращено -EINTR. Метод write блокирующий, в случае ошибки записи или
отсутствия метки будет возвращено отрицательное значение, в случае успеха - 
положительное.
	По умолчанию, драйвер находится в режиме чтения только UID метки,
метод read в данном случае вернет 16 байт первые 4(7) из которых будут ID метки,
остальные заполнены нулями. В случае перевода драйвера в нормальный режим, метод
read вернет данные из требуемого блока.
	Драйвер управляется по средствам ioctl. Всего предусмотрено 10 опций:
установить/получить номер блока (номер блока, над которым будут производиться
операции чтения/записи), установить/получить ключ А, установить/получить ключ Б,
установить/получить используемый ключ (А или Б), установить/получить режим работы
(чтение только UID/нормальный режим).
	Команды ioctl находятся в файле mfrc522_ioctl.h.
	MFRC522_IOCSBLADDR - устанавливает адрес блока
	MFRC522_IOCGBLADDR - считывает адрес блока
	MFRC522_IOCSKEYA - устанавливает ключ А (6 байт)
	MFRC522_IOCSKEYB - устанавливает ключ Б (6 байт)
	MFRC522_IOCGKEYA - считывает ключ А (6 байт)
	MFRC522_IOCGKEYB - считывает ключ Б (6 байт)
	MFRC522_IOCSKEY - устанавливает используемый ключ (А или Б, 0 - А, не 0 - Б)
	MFRC522_IOCGKEY - считывает используемый ключ (0 - А, 1 - Б)
	MFRC522_IOCSROID - устанавливает режим работы драйвера (0 - нормальный режим,
не 0 - режим чтения только UID)
	MFRC522_IOCGROID - считывает режим работы драйвера
